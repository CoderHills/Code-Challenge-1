Kenya Essentials JS Mini Projects
This repository contains three beginner-friendly JavaScript challenges designed to teach core programming concepts using familiar Kenyan scenarios. Each project interacts with users via prompts and outputs clear, friendly messages in the console.
 Projects Overview
1. Chai Bora Ingredient Calculator
A simple calculator that determines the ingredients needed to make a specific number of cups of Kenyan chai.

Features:
Prompts user for number of cups.

Calculates and displays:

Water (ml)

Milk (ml)

Tea Leaves (tablespoons)

Sugar (teaspoons)
Sample Output:
vbnet
Copy
Edit
To make 3 cups of Kenyan Chai, you will need:
Water: 600 ml
Milk: 150 ml
Tea Leaves (Majani): 3 tablespoons
Sugar (Sukari): 6 teaspoons

Enjoy your Chai Bora!
2.oda Boda Ride Fare Estimator
Estimates the cost of a boda boda ride based on trip distance.
Fare Formula:
Base Fare: KES 50

3. Mobile Money Transaction Fee Estimator
Calculates the transaction fee and total amount debited for sending money via mobile platforms.

 Fee Rules:
1.5% of transaction amount

Minimum fee: KES 10

Maximum fee: KES 70

Copy
Edit

Send Money Securely!
How to Run
Clone or download this repository.

Open any project file (chai-bora.js, boda-fare.js, mobile-fee.js) in a modern browser's developer console or use a JavaScript sandbox like JSFiddle or CodePen.

Each script will prompt you for input and show output in the console.
